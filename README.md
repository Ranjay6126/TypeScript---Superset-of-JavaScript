# TypeScript---Superset-of-JavaScript.

TypeScript - Superset of JavaScript

This repository contains examples and concepts of TypeScript, a powerful superset of JavaScript that adds static typing and advanced features to improve code quality and maintainability. It is designed for beginners and developers who want to transition from JavaScript to TypeScript.

🚀 Topics Covered

Introduction to TypeScript

TypeScript vs JavaScript

Setting up TypeScript

Number

Functions in TypeScript

Objects and Interfaces

Classes and OOP

Type Aliases

Union and Intersection Types

Generics

Modules

Type Inference

💡 Features

Beginner-friendly examples

Clean and simple code

Step-by-step learning

Real examples for practice

Easy to understand structure

🎯 Purpose

The main goal of this repository is to learn TypeScript from basics to advanced concepts and understand how it improves JavaScript development by adding type safety and better tooling support.
